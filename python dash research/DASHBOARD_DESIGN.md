📊 KEY METRICS DISPLAYED

Top Section (KPI Cards):
  • Total Leads: Count of all leads in selected filters
  • Pipeline Value (₹L): Total value of all leads
  • Active SBUs: Number of business units
  • Top Owner: Best performing owner with lead count
  • Stage Split: Planned vs Achieved leads
  • Top SBU: Best performing business unit

Chart Metrics:
  • Lead Status: Won/Lost/Pending/etc counts
  • Pipeline Stages: Leads in each stage
  • Customer Value: Revenue per top customers
  • Revenue Trend: Monthly revenue with forecast
  • Owner Productivity: Leads, wins, value per owner
  • SBU Performance: Status breakdown per business unit

🚀 HOW TO USE THE NEW DASHBOARD

1. LOAD THE DASHBOARD
   python app.py
   Open: http://localhost:8050

2. EXPLORE OVERVIEW
   • Look at KPI cards to understand data summary
   • Review executive dashboard (top 6 charts)
   • Identify key trends and outliers

3. APPLY FILTERS
   • Select one or more SBUs
   • Filter by Customer, Year, Lead Type
   • Watch all charts update automatically

4. DEEP DIVE ANALYSIS
   • Click "📊 Show Additional Metrics" for more charts
   • Analyze lost reasons
   • Review stage progression

5. EXPORT DATA
   • Click "📋 Show Raw Data"
   • Search for specific leads
   • Sort by any column
