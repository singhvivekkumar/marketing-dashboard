📊 THE 6 MAIN CHARTS - QUICK REFERENCE
═══════════════════════════════════════════════════════════════════════════════

1. DONUT CHART (Top Left)
   Purpose: Show lead status distribution at a glance
   Displays: Won, Pending, Lost, Others (with percentages)
   Color: Multi-color with clear separation
   Interactive: Hover for details, click legend to filter

2. FUNNEL CHART (Top Right)
   Purpose: Visualize sales pipeline progression
   Displays: Planned → Achieved → Closed stages
   Color: Blue gradient showing flow
   Interactive: See conversion between stages

3. HORIZONTAL BAR (Middle Left)
   Purpose: Identify top revenue-generating customers
   Displays: Top 10 customers by value (₹ Lakhs)
   Color: Green gradient (more value = darker)
   Interactive: Sort and hover for details

4. LINE CHART (Middle Right)
   Purpose: Track revenue trends and forecast future
   Displays: Monthly revenue + forecast line
   Color: Blue line with markers
   Interactive: Hover for monthly values

5. BUBBLE CHART (Bottom Left)
   Purpose: Evaluate individual owner performance
   Displays: Leads (X) vs Won (Y) vs Value (bubble size)
   Color: Viridis scale showing value
   Interactive: Hover for owner details

6. STACKED BAR (Bottom Right)
   Purpose: Compare business unit performance
   Displays: Each SBU with status breakdown
   Color: Won/Pending/Lost color-coded
   Interactive: See exact counts per status
