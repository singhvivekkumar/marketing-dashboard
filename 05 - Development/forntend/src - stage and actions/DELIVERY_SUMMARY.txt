🚀 QUICK START GUIDE
═══════════════════════════════════════════════════════════════════════════

Step 1: Access the Portal
  → Navigate to your app sidebar
  → Click "Bidding Management" under Portal section
  → Dashboard loads with 7 module cards

Step 2: Install Dependencies (if needed)
  → npm install @mui/material @mui/icons-material

Step 3: Configure Backend
  → Set REACT_APP_API_URL in .env
  → Point to your API: http://localhost:5000/api

Step 4: Replace Mock Data
  → Open views/LeadsModule.js
  → Replace sampleLeads with API call:
     leadService.getAllLeads().then(setLeads)

Step 5: Test Each Module
  → Click modules in sidebar
  → Verify data displays correctly
  → Test filters and search

Step 6: Deploy
  → npm run build
  → Deploy to production
